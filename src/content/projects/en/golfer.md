A personal side project: a web platform that introduces the sport of golf to newcomers.

## Features

- **Rules** — an approachable introduction to how golf is played.
- **Competitions** — overview of major tournaments and formats.
- **Stars** — profiles of notable players.

Built with **TypeScript**. Source code is available on [GitHub](https://github.com/seamuschang31/golfer).
